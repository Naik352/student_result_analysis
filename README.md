# student_result_analysis
Python Django web application for managing student records with authentication, database integration, and data visualization
Student Result Analysis System

A Django web application to manage student records and analyze academic performance.

## Features
- User login authentication
- Student dashboard
- Add, edit, delete student records
- Automatic result calculation
- Pass/Fail pie chart visualization

## Technologies
Python, Django, SQLite, Bootstrap

## How to Run
python manage.py runserver
